# AriGo - AirGuard Pro

Clean the air, Choose the care!

AriGo AirGuardPro is a portable wearable air purification system connected to this mobile application.

## Setup

1. Clone the repo
2. Add your own `google-services.json` from Firebase Console into the `app/` folder
3. Add your Google Maps API key in `AndroidManifest.xml`
4. Open in Android Studio and sync Gradle
5. Run on device/emulator

## Tech Stack

- Kotlin + Jetpack Compose
- MVVM + Clean Architecture
- Firebase Authentication
- Firebase Realtime Database
- Google Maps
- Room Database

## Note

`google-services.json` is excluded from this repo for security. You need to create your own Firebase project and add the config file.
